Data and attribution notes

The donor and outreach data in this project is adapted from "Fundraising Data" by Michael Pawlus, published on Kaggle, used here for educational and portfolio purposes. The original dataset is anonymized university fundraising data. I relabeled field names and categories to fit a food bank context. I do not claim ownership of the original dataset.

Source: https://www.kaggle.com/datasets/michaelpawlus/fundraising-data

The volunteer data in this project is fully synthetic, generated by me for this project. It does not represent any real person.

This project itself, meaning the workbook structure, formulas, dashboard design, and documentation, is free to look at, learn from, or reference for your own portfolio work. If you build something based on this, a credit back to this repo is appreciated but not required.
